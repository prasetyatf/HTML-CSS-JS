CSS: Cascading Style Sheets

Oleh: Tegar

Sumber CSS

CSS memiliki 3 jenis: Inline, External, Internal.

  • Inline menambahkan atribut pada elemen html.

  • Internal menambahkan CSS pada element <style> <⁄style> pada head.

  • External menambahakan elemen link yang merujuk pada file CSS.

Penulisan syntax CSS bergantung pada HTML. apakah itu element(p{}), id(#fname{}), class(.center{}). Penulisannya bisa spesifik di suatu elemen html seperti p.center{} atau spesifik berada di hirarki html seperti ul li{} atau grouping h1,h2,h3{harus memiliki declaration yang sama}

Specificity
Penentuan CSS Selector mana yang dieksekusi duluan berdasarkan hirarki di css, jika CSS selector memiliki nama yang sama, dan declaration yang sama, maka yang paling bawah yang dieksekusi.
Penentuan lain adalah dengan mengukur kadar spesifikasinya. Makin spesifik dan makin banyak, makin besar nilainya, vice versa.
Urutan dari yang paling spesifik: Inline css, id, class, element.

Layouting
Element/tag html digunakan untuk memberi arti pada sebuah konten, disebut semantic.
contoh: tag p = paragraf, tag h1 = heading utama. tetapi ada beberapa tag yang tidak memiliki
arti, seperti <div>, <span>. Dua tag itu digunakan untuk mengelompokkan tag-tag html lain.
Setiap tag html itu berada di dalam sebuah kotak, display pada CSS mengatur perilaku kotak tsb.
Semua tag html memiliki default display pada CSS(meski kamu tidak menulisnya), biasanya kalau tidak inline maka block, <div> memeiliki default display block, <span> memiliki display: inline.

Jenis display: inline, block, inline-block, none.
Elemen-elemen html dengan default display inline: b, strong, i, a, span, sub, sup, button, input, label, textarea, select.
Elemen-elemen html dengan default display block: div, p, h1, form, ul, li, table.
Elemen-elemen html dengan default display inline-block: tidak ada :p
Elemen-elemen html dengan default display none: style, script, meta, link

Contoh penggunaan block(violet) dan inline(blue violet)

Jenis-jenis Positioning:

  • position: static; -> default di semua element html. tidak berarti apapun(normal). Jika ada yang merubah position menjadi static, biasanya ingin menghapus efek apapun yang berlaku di elemen html dan di luar kendali kita.
  • position: relative; -> jika ditulis tanpa posisioning atribut, maka tidak berefek apapun. jika ditambah top: 10px;, maka itu akan menggeser posisinya 10 px ke bawah dari tempat biasanya. dan berefek pada atribut z-index dan child.
