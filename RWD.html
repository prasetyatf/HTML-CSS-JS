Responsive Web Design

Viewport

viewport adalah area yang dapat dilihat oleh user di suatu page html.
viewport ini dihandle oleh tag meta. tag meta ini memliki atribut content="width=device-width"
yang berarti witdh page html akan mengikuti width device. initial-scale menghandle zoom level saat page pertama x diload.

Aturan Ukuran Konten Untuk Viewport

Semua user di beberapa device menjelajah page secara vertikal(scrolling). Jadi ketika user dipaksa untuk scrolling horizontally atau perlu zoom untuk melihat keseluruhan page, maka it's poor user experience.
Aturan:

  1. Jangan guanakan fix width yang besar.

  2. Jangan biarkan konten mengandalkan lebar viewport tertentu untuk merender dengan baik.

  3. Lebih baik gunakan relativ width value daripada absoulut, dan hati-hati menggunakan absolut positioning value.

RWD Images

Best practice untuk css img adalah, menggunakan max-width=100% dan max-height=auto.
Bisa juga gunakan element <picture> untuk RWD Images.
Untuk background image perhatikan opsi apa saja yang ada di css background-size(persentase, contain, cover).
img di bawah ini menggunakan css img RWD.

img di bawah ini menggunakan element picture.
